# Requirements:
- You will need a Spotify API key, and to create a .env file in which to store it in. like so:
```
# Spotify API keys

SPOTIFY_ID=
SPOTIFY_SECRET=
```
- You will also need to install dependencies as needed via the terminal:
```
npm i
```
or
```
npm install
```
